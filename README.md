## Description
LoginOTPApp provides a secure login system for mobile apps with a "Forgot Password" feature using **OTP verification**.  
It demonstrates a complete full-stack architecture: mobile frontend, backend API, and database integration.

---

## Features
- Secure user login with **encrypted passwords**
- Forgot Password functionality with **OTP generation**
- OTP verification and **password reset**
- Backend REST APIs using **Spring Boot**
- MySQL database storage for user data and OTP
- Clean, modular React Native frontend
- Easily extendable and maintainable structure

---

## Tech Stack
- **Frontend:** React Native, Axios  
- **Backend:** Java, Spring Boot, Spring Security, JPA/Hibernate  
- **Database:** MySQL  
- **Tools:** Node.js, npm, Git/GitHub
